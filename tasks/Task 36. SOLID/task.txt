Сервис предоставляет возможность заказа такси. 2 типа пользователей:  клиент – может размещать заказы на поездки с выбранным типом комфорта, цена формируется автоматические исходя из километража (можно брать из API карт); водитель – может выполнять заказы на поездку. Также должна быть возможность написания отзыва по окончанию поездки.
